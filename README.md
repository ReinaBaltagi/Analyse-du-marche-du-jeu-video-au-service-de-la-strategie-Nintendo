#Analyse du marché du jeu vidéo au service de la stratégie Nintendo
Cette étude analyse les ventes mondiales de jeux vidéo (1980–2020) pour identifier les genres, plateformes et tendances les plus porteurs. Elle fournit à Nintendo des pistes stratégiques pour guider le développement de son prochain jeu exclusif et saisir les opportunités du marché.

#Contexte: 
Nintendo souhaite développer un nouveau jeu vidéo exclusif et envisage d’explorer de nouvelles directions créatives : univers inédits, thématiques innovantes ou encore nouveaux types de gameplay.
Avec un jeu de données consolidé sur les ventes mondiales de jeux vidéo entre 1980 et 2020, nous allons fournir une étude de marché détaillée

#Objectif:
Identifier, à partir de l’analyse de ces données, des pistes stratégiques pour orienter la conception du prochain jeu vidéo Nintendo.

#Problématique

Quelles recommandations peut-on proposer à Nintendo concernant :

les genres les plus porteurs,

les plateformes les plus performantes,

les tendances historiques du marché,

les régions les plus prometteuses,

ou encore les opportunités d’innovation ?
